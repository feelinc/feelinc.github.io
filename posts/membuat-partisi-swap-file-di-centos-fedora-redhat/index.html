Membuat partisi SWAP file di CentOS / Fedora / RedHat

2017-03-08 :: Sulaeman
#cloud  #server 

Saya harap Disk volume yang akan dijadikan tempat penyimpanan swapfile sudah tersedia

Putuskan ukuran swap yang akan dibuat, misal 4 GB = 1024 * 4.

Putuskan direktori yang akan dijadikan tempat membuat swapfile, misal /mnt/data/swapfile.

Selanjutnya, setelah Anda telah masuk ke console VM Instance, gunakan perintah – perintah berikut ini:

sudo dd if=/dev/zero of=/mnt/data/swapfile bs=4096 count=1MiB

Tunggu hingga selesai, lumayan lama tergantung ukuran swapfile yang sedang dibuat.

Selanjutnya, gunakan perintah – perintah berikut ini:

sudo chmod 600 /mnt/data/swapfile

sudo mkswap /mnt/data/swapfile

sudo swapon /mnt/data/swapfile

Silahkan periksa apakah partisi Swap sudah aktif menggunakan perintah : free -m

Supaya ketika server restart partisi Swap akan otomatis aktif, maka harus mendaftarkannya di fstab. Edit /etc/fstab, kemudian tambahkan :

/mnt/data/swapfile                        swap                    swap    sw              0 0
